# AAIDES Fish Detector — Clean Firebase Auth Version

This project is a lightweight React + Vite starter with Firebase Email/Password authentication.
It uses normal CSS (no Tailwind) to avoid PostCSS issues and works on Windows/VSCode reliably.

## Setup

1. Extract the ZIP and open folder in VS Code.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
4. Make sure Firebase Auth Email/Password is enabled in your Firebase console.
5. (Optional) Paste your model endpoint into `src/api.js` and implement prediction UI.
